Bilingual Files
===============

Bilingual files are files that are in the process of translation.
:ref:`KXLIFF<KXLIFF>` files are created by kaplan while :ref:`XLIFF<XLIFF>` and
:ref:`SDLXLIFF<SDLXLIFF>` files are created elsewhere.
